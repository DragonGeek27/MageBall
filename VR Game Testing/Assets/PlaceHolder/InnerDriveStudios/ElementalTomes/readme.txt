This package contains 4 elemental tomes, in 3 different quality settings.
Note that all textures, irrespective of the quality settings have been exported at 2k, 
but have different export settings in Unity based on their expected usage.

For example:
High quality materials -> max texture size 2k
Medium quality materials -> max texture size 1k
Low quality materials -> max texture size 512

If you want to improve the texture quality for the medium and low quality materials,
increase their max texture size setting to 2k.
